# 3AM Frontend

Frontend for the 3AM car showcase and accessories store. Built as a small SPA with a lightweight `View` system,
sections, and pages.

[Backend repository](https://github.com/ali7510/3AM-Project)

## Documentation

VitePress documentation site lives in `docs/`.

Run locally:

```bash
bun run docs:dev
```

Build docs:

```bash
bun run docs:build
```

GitHub Pages docs deployment runs from:

`/.github/workflows/docs-pages.yml`

In repository settings, set Pages source to **GitHub Actions**.

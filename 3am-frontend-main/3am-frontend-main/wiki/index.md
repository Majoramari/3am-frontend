# 3AM Frontend Guide

> Note: This repo is frontend only. Backend: [ali7510/3AM-Project](https://github.com/ali7510/3AM-Project)

The UI is built with `View` classes. Components, sections, and pages are all `View` subclasses that render into the DOM
and clean up after themselves.

## Read First

1. [Components](components.md) Reusable pieces.
2. [Sections](sections.md) Page-only blocks.
3. [Pages](pages.md) Routing and page layout. Includes a cart walkthrough.
4. [Templates](templates.md) How to write markup with `html` and `this.tpl`.
5. [Styling Guide](styles.md) Where styles live and how to add them.
6. [Guidelines](guidelines.md) Rules to keep code consistent.
7. [Contributing](contributing.md) Fork, edit, and open a pull request.

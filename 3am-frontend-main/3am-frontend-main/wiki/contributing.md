# Contributing

Use this flow when contributing to the project:

1. Check the [project plan](https://github.com/users/Majoramari/projects/12/views/2) before starting any work.
2. Add what you will do to the project plan.
3. Assign the plan item to yourself or to another teammate.
4. Do not work on plan items assigned to someone else unless they are reassigned.
5. Fork this [repository](https://github.com/Majoramari/3am-frontend).
6. Clone your fork and create a branch for your change.
   ```bash
   git clone https://github.com/<your-github-username>/3am-frontend.git
   cd 3am-frontend
   git checkout -b my-change
   ```
7. Edit the code and commit your updates.
   ```bash
   git add .
   git commit -m "feat: describe change"
   ```
   Note: Follow [Conventional Commits](https://www.conventionalcommits.org/) for commit messages.
8. Push your branch to your fork.
   ```bash
   git push -u origin my-change
   ```
9. Open a pull request to this [repository](https://github.com/Majoramari/3am-frontend).

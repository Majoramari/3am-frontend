# 3AM Docs

Welcome to the engineering docs for **3AM**.

The repository name is `3am-frontend`, but the project name is **3AM**. This documentation is written for contributors
who need to understand the code quickly, make safe changes, and open clean pull requests.

## Why this documentation exists

These docs are not a wiki dump. The goal is practical: when you open a page, you should understand what to edit, why it
is done that way in this project, and how to validate your change before PR.
